How to add slides

  1. Duplicate a full <section class="slide">...</section> block.
  2. Change the slide’s data-title and data-section attributes.
  3. Replace the HTML between SLIDE BODY START and SLIDE BODY END.
  4. Add optional references inside the hidden .slide-references block.
  5. Use an iframe slide for full demos and the inline demo box for smaller snippets.

Keyboard controls

  • → or Space: next slide
  • ←: previous slide
  • Home / End: first / last slide
  • T: open slide list
  • R: open references for current slide
  • F: fullscreen

Math example Kernel methods

Kernel ridge regression

We solve a regularized least-squares problem in an RKHS.

Objective

Given training data \((x_1,y_1),\dots,(x_n,y_n)\), we minimize

\[ \min_{f \in \mathcal H} \sum_{i=1}^n \bigl(f(x_i)-y_i\bigr)^2 + \lambda \|f\|_{\mathcal H}^2 . \]

By the representer theorem, the solution has the form

\[ f(\cdot)=\sum_{i=1}^n \alpha_i K(x_i,\cdot). \]

Closed-form coefficient solution

Let \(K \in \mathbb R^{n\times n}\) be the kernel matrix, where \(K_{ij}=K(x_i,x_j)\). Then

\[ \alpha = (K+\lambda I)^{-1} y. \]

Prediction at a new point \(x\) is

\[ \hat f(x)=\sum_{i=1}^n \alpha_i K(x_i,x). \]
