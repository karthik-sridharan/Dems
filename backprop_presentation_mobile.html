The animation arc for the lecture

Scene 1

Two inputs, two hidden neurons, one output. Introduce the question: which weights caused the final loss?

Scene 2

Step 0

The tiny network we will study

Start with the smallest network that still contains the full backprop story.

Inputs

x₁ and x₂ are the features that enter the network.

Hidden layer

h₁ and h₂ let us show how error is assigned to intermediate units.

Output

ŷ is the network prediction.

Target + loss

We compare ŷ to y using a scalar loss.

z^(1) = W^(1)x + b^(1) a^(1) = σ(z^(1)) z^(2) = W^(2)a^(1) + b^(2) ŷ = a^(2)
Step 1

Forward pass: compute hidden activations

Blue means information moving forward through the network.

z^(1) = W^(1)x + b^(1) a^(1) = σ(z^(1))
  • Each hidden neuron first computes a weighted sum z.
  • Then it applies a nonlinearity to produce an activation a.
  • These values are cached because the backward pass will reuse them.
cache z^(1) cache a^(1)
Step 2

Forward pass: compute the prediction

z^(2) = W^(2)a^(1) + b^(2) ŷ = a^(2)
  • The hidden activations now act like the inputs to the output layer.
  • The network emits a prediction ŷ.
  • We still have not learned anything yet — we have only evaluated the current parameters.
Step 3

Compare prediction to target

L = ½(ŷ − y)²
  • The forward pass ends in a single scalar: the loss.
  • If the prediction changes, the loss changes.
  • That sensitivity is what the backward pass will propagate.
Backward pass

Start at the loss

The first backward question is simple: if the prediction moved a little, how would the loss move?

∂L/∂ŷ
  • This quantity is the sensitivity of loss to the prediction.
  • It is the seed that begins reverse accumulation through the graph.
  • From now on, red means gradient flow.
Local chain rule

Backprop through the output neuron

∂L/∂z^(2) = (∂L/∂ŷ) · (∂ŷ/∂z^(2))
  • The first factor is the incoming gradient from downstream.
  • The second factor is the neuron’s local slope.
  • This exact multiplication pattern repeats at every stage of backprop.
incoming gradient × local derivative = new gradient
Output layer gradients

Assign credit to the output weights

∂L/∂w_j^(2) = (∂L/∂z^(2)) · a_j^(1)
  • A weight gets a large gradient when a strong error signal reaches it.
  • It also gets a large gradient when the activation flowing through that edge is large.
  • That is why the output weights are easy to interpret: error signal × incoming activation.
Hidden layer backprop

Send responsibility back to the hidden layer

∂L/∂a^(1) = (W^(2))ᵀ · (∂L/∂z^(2)) ∂L/∂z_j^(1) = (∂L/∂a_j^(1)) · σ'(z_j^(1))
  • The output error signal splits across the hidden units.
  • Each hidden neuron receives a weighted share of downstream responsibility.
  • Then it multiplies by its own local derivative before passing anything farther back.
First-layer gradients

Now compute gradients for earlier weights

∂L/∂w_jk^(1) = (∂L/∂z_j^(1)) · x_k
  • The local error signal at hidden unit j is already known.
  • Multiply it by the input that crossed the edge, xₖ.
  • Backprop is recursive: the same local rule keeps reappearing.
local error signal × input activation
Learning step

Turn gradients into a parameter update

w ← w − η ∇_w L
  • Once all gradients are available, gradient descent adjusts the parameters.
  • The learning rate η sets how far we move.
  • Then we run another forward pass and hope the loss is lower.
Before

current weights produce current loss

After

updated weights should reduce loss on the next pass

One-picture summary

The backprop recipe to remember

1. Run forward

Compute z and a values through the network and cache them.

2. Run backward

Start at the loss, then repeatedly apply incoming gradient × local derivative.

3. Update parameters

Use gradient descent or another optimizer to move weights in a better direction.

Forward: x → z^(1) → a^(1) → z^(2) → ŷ → L Backward: ∂L/∂ŷ → ∂L/∂z^(2) → ∂L/∂a^(1) → ∂L/∂z^(1) → ∂L/∂W

Backpropagation is just the chain rule organized efficiently over the computation graph.

Optional extension

Why gradients can vanish in deep networks

If many local derivatives are smaller than one, multiplying them repeatedly can make the backward signal shrink.

small × small × small × small → tiny gradient
  • This is one reason saturated sigmoid layers can be hard to train when stacked deeply.
  • It motivates careful initialization, activation choices, and architectural tricks.
