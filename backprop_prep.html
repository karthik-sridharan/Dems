A Multi-Layer Fully Connected Neural Network
\[ y = \alpha^\top \sigma\!\left(W^{[2]}\sigma\!\left(W^{[1]}x\right)\right) + b \]
ReLU activations lead to piecewise linear models.
Key idea

A deep fully connected network alternates between:

  • linear maps
  • coordinatewise nonlinearities
  • a final output layer
The lecture uses this setup to motivate both the forward pass and the backward pass.

Why Go Deep?
  • A sufficiently wide network with one hidden layer can represent any piecewise linear function.
  • Depth can greatly reduce the width needed for the same function class.
Extra depth can buy expressive power more efficiently than only making a network wider.

Forward Pass
\[ z^0 = x,\qquad u^t = W^{[t]}z^{t-1} + b^t,\qquad z^t = \sigma(u^t),\qquad \hat y = \alpha^\top z^{[T]} + b \]
\(z^0=x\)
→
\(u^1=W^{[1]}z^0+b^1\)
→
\(z^1=\sigma(u^1)\)
→
⋯
→
\(z^{[T]}=\sigma(u^{[T]})\)
→
\(\hat y\)
Computation pattern
  • Start at the input layer.
  • Compute every neuron’s output once.
  • Move layer by layer until the final prediction.
Forward-pass time is linear in the total number of nodes plus edges in the computation graph.

Training Neural Networks via SGD
\[ h(x) := \alpha^\top \sigma\!\left(W^{[2]}\sigma\!\left(W^{[1]}x\right)\right) + b \]
Goal
Let \(\ell(h(x), y)\) be any differentiable loss function.
Compute gradients
\[ \frac{\partial \ell(h(x),y)}{\partial W^{[1]}} \qquad \frac{\partial \ell(h(x),y)}{\partial W^{[2]}} \] \[ \frac{\partial \ell(h(x),y)}{\partial \alpha} \qquad \frac{\partial \ell(h(x),y)}{\partial b} \]

Mini-Batch Stochastic Gradient Descent
\[ \theta = [W^{[1]}, W^{[2]}, \alpha, b] \]
For epoch \(e = 1,2,\dots,E\):
Randomly shuffle the dataset.
Split the data into \(n/B\) mini-batches \(\mathcal{D}_i\), each of size \(B\).
For \(j = 1,2,\dots,n/B\):
Mini-batch gradient \[ g = \frac{1}{B}\sum_{(x,y)\in \mathcal{D}_i}\nabla_\theta \ell(h_\theta(x),y) \]
\[ \theta \leftarrow \theta - \eta g \]
Go through the dataset multiple times.
Mini-batch gradients are unbiased estimates of the full gradient.

Backpropagation
Backward pass: compute gradients using intermediate variables from the forward pass.
\[ \nabla_\alpha \ell(\hat y, y) = \ell'(\hat y, y)\, z^{[T]}, \qquad \nabla_b \ell(\hat y, y) = \ell'(\hat y, y) \] \[ \delta^{[T]} = \ell'(\hat y, y)\,\alpha \odot \sigma'(u^{[T]}) \]
For \(t=T,T-1,\dots,1\):
\[ \nabla_{W^{[t]}}\ell(\hat y, y) = \delta^{t}(z^{t-1})^\top \] \[ \nabla_{b^t}\ell(\hat y, y) = \delta^t \] \[ \delta^{t-1} = \sigma'(u^{t-1}) \odot W^{[t]}\delta^t \]
Reverse-mode differentiation through the network

Optimization Tricks
  • Use momentum.
  • Reduce the learning rate gradually over time.
  • Use small mini-batches.
  • Scale features into \([0,1]\) or another consistent range.
  • Initialize weights randomly from a suitably scaled normal distribution.
Practical tips for training stability
9 / 13
Batch Normalization
  • For each layer, compute batch statistics over the mini-batch: \[ \mu^t = \frac{1}{B}\sum_{(x,y)\in \mathcal D_i} u^t(x,y) \] \[ \sigma^t = \text{variance of }u^t\text{ over the mini-batch} \]
  • Normalize and re-scale: \[ \mathrm{BN}(u^t) = \gamma_t \frac{u^t-\mu_t}{\sqrt{\sigma^t+\varepsilon}} + \beta_t \]
  • Then apply the activation: \[ z^t = \sigma(\mathrm{BN}(u^t)) \]
  • \(\gamma_t\) and \(\beta_t\) are learned parameters.
  • Batch normalization can speed up training and reduce internal covariate shift.

Residual Connections
  • Add the input back to the output activations after a layer.
  • Learning the identity map \(\phi(x)=x\) directly can be difficult.
  • Residual connections help with vanishing gradients.
Skip connections for easier optimization

Dense Connections
  • Concatenate the input with the output activations after a layer.
  • This keeps earlier features available to later layers.
