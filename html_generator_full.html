How to write content

Use plain text with light structure:

- Start lines with - for bullets.

- Or use prefixed lines: P:, UL:, EQ:, CARD:title|body.

- Start lines with ### for a small section heading.

- Leave blank lines between paragraphs.

- Inline math: $x^2$

- Displayed math: put \( ... \) on its own line, or use \[ ... \].
