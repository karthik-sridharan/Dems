HTML slide snippet generator

Fill in the form, preview the result, then copy the generated <section class="slide">...</section> block into one of your existing HTML presentations.

Use one item per line. Prefixes: P:, UL:, EQ:, CARD:title|body. Plain text lines become paragraphs. Math is supported with $x$ for inline math and \( x^2 + y^2 \) for displayed math.
If the mode is panel, this uses the same line prefixes as the left column. If the mode is custom, paste raw HTML. Inline math like $\nabla L$ and displayed math like \( \frac{dL}{dw} = x \) will render in the preview.
How to use the generated code

Paste the generated <section class="slide" ...> block into your existing HTML deck inside the main .deck container. The snippet uses the same CSS class names as the earlier presentation files.

Good workflow on phone: use Copy snippet for your deck, or use Download standalone .html to export a self-contained single-slide page that should match the preview.

The color menus add inline slide styles, so the copied snippet keeps your chosen background and font colors when pasted into the larger deck.

This version removes chips entirely and keeps kicker optional, with kicker shown below the title when used.

Math preview uses MathJax. If your deck does not already load MathJax, tap Copy MathJax helper once and paste that script block near the end of your deck HTML.
