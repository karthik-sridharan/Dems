Backpropagation on a \(2 \times 3 \times 1\) Network

Each hidden unit is split into a linear node \(z_j\) and an activation node \(a_j=\sigma(z_j)\), and the output is split into a linear node \(s\) and prediction node \(\hat y\). This makes it visually clear where the factor \(\sigma'(z_j)\) enters during backpropagation.
